# General Information
## What is {{ site.title }}?
Nutri-Matic is a collection of tools that facilitates Cookiecutter template projects and exists inside of the deployed templates to facilitate CI/CD processes.  

## Why was {{ site.title }} created in the first place?
 Nutri-Matic initially was going to be used to more easily deploy Cookiecutter projects with custom json/yaml configurations used to replace the manual setup of a template. It was also intended to drop custom documentation templates into existing projects.  

## Why is it called {{ site.title }}?
Nutri-Matic is named after [The Hitchhiker's Guide to the Galaxy](https://en.wikipedia.org/wiki/The_Hitchhiker%27s_Guide_to_the_Galaxy) Nutrimatic Drinks Dispenser.
